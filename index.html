LearnOptimus 리포트 시스템

2.1 학습자 리포트

  • 제공 대상: LearnOptimus에 등록된 학습자
  • 제공 시점: 매일, 평가 완료 후, 학습 경로 완료 후
  • 제공 방법: 대시보드 내 실시간 표시, 이메일 발송(선택)
  • 목표: 학습 경로 진행 상태와 성과 분석을 통해 학습자가 스스로 성찰하고 학습을 개선할 수 있도록 지원
리포트 종류 주요 내용 제공 시점
학습 경로 진행 리포트 학습 경로의 각 단계별 진행률, 예상 완료 시간, 보충 학습 추천 매일 또는 학습 경로 변경 시
성취도 리포트 과목별 성취도, 성과에 대한 AI 분석, 학습 강점과 약점 평가(퀴즈, 시험) 완료 후
피드백 리포트 학습 오류에 대한 설명, 보충 학습 권장 자료 학습 단계 완료 후
자기 평가 리포트 학습자가 설정한 목표, 강점 및 어려움에 대한 자기 성찰 결과 매주 또는 매달
종합 학습 성과 리포트 전체 학습 경로 성취도 요약, 성과 분석, 향후 학습 계획 제안 학습 경로 완료 시

2.2 교수자 리포트

  • 제공 대상: LearnOptimus에서 학습자 그룹을 관리하는 교수자
  • 제공 시점: 매주, 학습 경로 완료 후, 평가 완료 후
  • 제공 방법: 대시보드 내 실시간 표시, 이메일 발송(선택)
  • 목표: 학습자 개개인의 성과와 학습 경로 진행 상태를 모니터링하고, 맞춤형 피드백을 제공하여 학습 효과를 극대화
리포트 종류 주요 내용 제공 시점
학습자 성취도 리포트 개별 학습자 성취도 및 진행 상태, 학습 강점 및 약점 분석 매주 또는 특정 학습 단계 완료 후
전체 학습 경로 성과 리포트 학급 또는 그룹 전체의 성취도, 평균 성취도, 각 학습 단계별 완료율 학습 경로 단계 완료 시 또는 매달
평가 분석 리포트 평가 결과에 대한 분석, 성공률 및 학습자별 성과, 오답 분석 평가 완료 후
교수자 피드백 리포트 교수자가 제공한 피드백 요약, 피드백 후 학습자의 성과 변화 피드백 제공 후
AI 추천 학습 경로 리포트 AI 기반 학습 경로 추천 및 학습 패턴 분석, 학습 경로 수정 제안 학습 경로 변경 시

2.3 학부모 리포트

  • 제공 대상: LearnOptimus에 등록된 학습자의 학부모

리포트 종류 주요 내용 제공 시점
학습 진행 리포트 자녀의 학습 경로 진행 상태, 각 단계별 완료율 매주 또는 학습 경로 변경 시
성과 분석 리포트 과목별 성취도, 성취도 대비 자녀의 강점과 약점 분석 매달 또는 평가 완료 후
평가 결과 리포트 자녀가 수행한 평가의 결과 요약 및 오답 분석 평가 완료 후
성취도 추적 리포트 자녀의 성취도 변화 추이, 장기적 성과 추적 학습 경로 완료 시 또는 학기 말
학습 목표 달성 리포트 자녀의 학습 목표 달성 여부, 목표 달성률 및 보충 필요성 학습 목표 완료 시
